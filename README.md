**Description**
An institutional Open Ledger Technology based on the Ethereum blockchain to facilitate transparency in transactions.

**How to install**
npm install

**Dependencies**
-Truffle suite
-Metamask

**How to run**
- Install truffle suite and start up the the truffle blockchain testnet.
- Connect metamask to the test blockchain and use any of the keys pairs provided by truffle to run the project
- cd into React web app and run npm start
